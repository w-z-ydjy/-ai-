# 智慧党建平台 - 页面分解说明

## 项目概述
本项目将原来的单一大型SPA（单页应用）HTML文件拆分为多个独立的HTML页面，每个页面都可以独立运行。

## 文件结构
```
pages/
├── styles.css          # 共享样式文件
├── common.js           # 共享JavaScript函数
├── index.html          # 首页
├── login.html          # 登录页
├── party.html          # 党建引领专区
├── party-detail.html   # 文章详情页
├── showcase.html       # 社团风采
├── showcase-detail.html # 社团风采详情
├── activities.html     # 党建活动
├── activity-detail.html # 活动详情
├── study.html          # 学习中心
├── quiz.html           # 在线答题
├── chat.html           # AI智能问答
├── message.html        # 留言板
├── profile.html        # 个人中心
└── admin.html          # 后台管理
```

## 各页面功能

### 首页 (index.html)
- 轮播图展示
- 快速入口导航
- 党建要闻列表
- 入党誓词展示
- 统计数据展示
- 近期活动展示

### 登录页 (login.html)
- 用户登录（admin/123456 或 user/123456）
- 访客登录模式

### 党建引领专区 (party.html)
- 党建新闻
- 政策文件
- 党史学习
- 党章党规
- 红色文化
- 科技强国专栏

### 社团风采 (showcase.html)
- 社团简介
- 党建+AI活动
- 成员风采
- 荣誉成果

### 党建活动 (activities.html)
- 活动列表展示
- 活动筛选（全部/报名中/已结束）
- 报名入口

### 学习中心 (study.html)
- 学习资料下载列表
- 在线答题入口
- 团课学习
- 学习排行榜
- 学习记录

### AI智能问答 (chat.html)
- AI党建助手对话界面
- 常见问题快捷入口
- 实时对话功能

### 留言板 (message.html)
- 留言提交表单
- 已审核留言展示

### 个人中心 (profile.html)
- 用户信息展示
- 报名记录
- 学习记录
- 我的收藏
- 个人成就/徽章
- 资料下载记录
- 个性化设置

### 后台管理 (admin.html)
- 控制台概览
- 用户管理
- 党建内容管理
- 活动管理
- 资料管理
- 留言审核

### 在线答题 (quiz.html)
- 党建知识测验
- 单选题、多选题、判断题
- 答题进度显示
- 答题结果统计

## 使用说明

1. **直接打开**：双击 `index.html` 即可在浏览器中打开首页
2. **页面跳转**：各页面之间通过链接跳转
3. **登录测试**：
   - 管理员：admin / 123456
   - 普通用户：user / 123456

## 技术特性

- **响应式设计**：适配桌面端和移动端
- **深色/浅色主题**：支持主题切换
- **字体大小调节**：支持字体大小个性化设置
- **本地存储**：使用 localStorage 保存用户设置和状态

## 注意事项

1. 部分功能（如文件下载、AI真实对话）需要后端支持
2. 当前AI问答使用本地知识库作为兜底方案
3. 所有页面共享 `styles.css` 和 `common.js`
